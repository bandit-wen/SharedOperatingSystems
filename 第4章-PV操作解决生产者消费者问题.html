🏭 生产者 (Producer)
while (true) {
// 生产数据 item
P(Sin); // 申请一个空槽位
P(mutex); // 互斥访问缓冲区
// 放入产品到 buffer
V(mutex); // 释放互斥锁
V(Sout); // 已占用产品数+1
}
⏩ 当前步骤: 等待执行
🍽️ 消费者 (Consumer)
while (true) {
P(Sout); // 申请一个产品
P(mutex); // 互斥访问缓冲区
// 从 buffer 取出产品
V(mutex); // 释放互斥锁
V(Sin); // 空槽位+1
// 消费产品
}
⏩ 当前步骤: 等待执行
📤 Sin (可放槽位)1
📥 Sout (可取产品数)0
🔒 mutex (互斥)1
点击按钮，单步执行伪代码行
💡 单步执行模式：每次点击只执行一条语句。可调整缓冲区大小（1~6），环形缓冲区动态显示。
