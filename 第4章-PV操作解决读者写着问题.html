📚 读者-写者问题 (PV操作 · 单步执行)

读者优先策略 | 可并发读，写者互斥 | 单步执行伪代码
📖 读者 (Reader)
// 读者进程
P(mutex);
readCount++;
if (readCount == 1)
P(db);
V(mutex);
// 读数据 (无操作)
P(mutex);
readCount--;
if (readCount == 0)
V(db);
V(mutex);
⏩ 当前步骤: 等待执行
✍️ 写者 (Writer)
// 写者进程
P(db);
// 写数据 (修改共享资源)
V(db);
⏩ 当前步骤: 等待执行
